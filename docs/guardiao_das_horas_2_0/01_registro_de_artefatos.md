# GDH2 — Registro de Artefatos no Canvas

Este documento é o **índice** dos artefatos gerados nas discussões do Guardião das Horas 2.0.
A “fonte de verdade” é o **Documento Mestre** (Canvas: *Guardião das Horas 2.0 — Documento Mestre*), e os demais itens abaixo são **anexos editáveis** para rastreio, revisão e evolução.

## Como usar
- Atualizações de regras/requisitos: edite o anexo correspondente **e** reflita o resumo no Documento Mestre.
- Mudanças estruturais (ERD/migrations): atualize **ERD v2**, **Plano Alembic** e, se necessário, **Modelo Físico**.
- Código “pronto pra colar”: mantenha em **Alembic Revision Templates**.
